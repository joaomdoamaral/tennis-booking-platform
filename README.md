# Tennis Booking Platform

Fullstack platform for booking tennis lessons and courts.

## Objective

This project was created as a portfolio application to simulate a real tennis scheduling platform with authentication, user roles, availability management, bookings, cancellations, simulated payments, waitlist and administrative dashboard.

## Stack

### Backend
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA
- PostgreSQL
- Flyway
- JUnit / Mockito

### Frontend
- Angular
- Reactive Forms
- Guards
- HttpInterceptor JWT

### Infrastructure
- Docker
- Docker Compose
