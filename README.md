# Tennis Booking Platform

Fullstack platform for booking tennis lessons and courts.

## Stack
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA
- PostgreSQL
- Flyway
- Angular
- Docker
- JUnit / Mockito

## Roles
- Admin
- Coach
- Student

## Modules
- Auth
- Courts
- Coaches
- Students
- Availability
- Reservations
- Payments
- Waitlist
- Dashboard
